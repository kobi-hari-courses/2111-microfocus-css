Simple Selectors

Type Selector (name selector)

            h3 - selects all elements of type h3

Class Selectors

            .class-name - selects all elements that contain the class 'class-name'

Id Selector

            #id-name - selects "all" (single) element with the id '#id-name'

Universal Selector

            * - selects all the elements
